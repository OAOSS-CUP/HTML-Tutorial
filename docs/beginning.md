## 软件依赖

你需要有以下两个软件

伟大的 [Chrome](https://www.google.cn/intl/zh-CN/chrome/) 以及神圣的 [VScode](https://code.visualstudio.com/)

## VScode 插件

我强烈建议你安装至少以下几个插件

| 插件名 | 用途 |
| --- | ---|
| Chinese (Simplified)（简体中文） | VScode中文适配 |
| open in browser | 在浏览器中查看HTML |
| Code Spell Checker | 检查拼写错误 |

## VScode 快捷键

我强烈建议你学会以下快捷键

| 快捷键 | 用途 |
| --- | ---|
| Ctrl+S | 保存 |
| Ctrl+F | 查找 |
| Ctrl+H | 替换 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+A | 全选 |
