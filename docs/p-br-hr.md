# 段落、换行、水平线

</div>

## 段落

```p``` 定义了段落

```html
<p>一个段落</p>
```

## 换行

```br``` 用于换行（不产生新段落）

```html
<p>一个段落</p>
<br>
```

## 水平线

```hr``` 可以添加一条水平线

```html
<hr color="red" width="300px" height="300px" align="center">
```

```color``` 定义颜色

```width``` 定义长度

```height``` 定义高度

```align``` 定义位置，默认居中（可以不写）

最简单的写法是

```html
<hr>
```
